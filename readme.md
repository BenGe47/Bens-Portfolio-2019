# Read Me Ben's Portfolio
Portfolio collection of cellular automaton, L-system, Random-Tree art and other generators.

## Getting Started
+ Run in Netbeans directly or as imported zip.
+ Run PortfolioMenuTest.jar directly.

### Prerequisites
+ JRE 1.8+

##Built With
+ SDK 1.8
+ Netbeans 8.2
+ Eclipse Neon 3
